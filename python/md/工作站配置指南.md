## 🚀 訓練命令

### 選項 A: 只使用 COCO 訓練

```bash
conda activate sgla

# 單 GPU
python tracking/train.py \
  --script sglatrack \
  --config coco_only \
  --save_dir ./output \
  --mode single \
  --use_wandb 0

# 多 GPU (例如 4 張)
python tracking/train.py \
  --script sglatrack \
  --config coco_only \
  --save_dir ./output \
  --mode multiple \
  --nproc_per_node 4 \
  --use_wandb 0
```

## ⚠️ 重要注意事項

### 1. **COCO 標註檔案**
確保解壓了 annotations：
```bash
cd data/coco/
ls annotations/
# 應該看到: instances_train2017.json, instances_val2017.json
```

### 2. **訓練時間估計**
只使用 COCO 訓練：
- COCO2017 有約 118k 訓練圖片
- 每個 epoch 60000 個樣本
- 300 epochs 大約需要 2-4 天（取決於 GPU）

### 3. **記憶體需求**
- Batch size 32: 需要約 11GB GPU 記憶體
- 如果記憶體不足，可以減少 batch size：
  ```yaml
  TRAIN:
    BATCH_SIZE: 16  # 或 8
  ```

### 4. **驗證頻率**
原始配置驗證很少（999 epochs 才驗證一次）。
建議改為：
```yaml
TRAIN:
  VAL_EPOCH_INTERVAL: 20  # 每 20 個 epoch 驗證一次
```
