# 📊 圖表格式說明與調整

## 📝 `.tex` 檔案是什麼？

### 用途
`.tex` 檔案是 **LaTeX 格式的圖表檔案**，用於學術論文撰寫。

**生成位置**:
```
output/test/result_plots/uav123/
├── success_plot.tex           - LaTeX 格式的 Success Plot
├── precision_plot.tex         - LaTeX 格式的 Precision Plot  
├── norm_precision_plot.tex    - LaTeX 格式的 Norm Precision Plot
├── success_plot.pdf           - PDF 格式（一般使用）
├── precision_plot.pdf         - PDF 格式（一般使用）
└── norm_precision_plot.pdf    - PDF 格式（一般使用）
```

### 為什麼需要 `.tex` 檔案？

#### 優點
1. **論文整合**: 可以直接嵌入 LaTeX 論文中
2. **一致性**: 與論文其他部分的字體、格式保持一致
3. **高品質**: 向量圖形，無限縮放不失真
4. **可編輯**: 可以在 LaTeX 中進一步調整

#### 使用場景
- 撰寫學術論文（IEEE、Springer、ACM 等格式）
- 需要與論文文字保持一致的字體
- 需要在 LaTeX 中調整圖表細節

### 如何使用 `.tex` 檔案？

在 LaTeX 論文中：
```latex
\begin{figure}[htbp]
    \centering
    \input{figures/success_plot.tex}
    \caption{Success plot on UAV123 dataset.}
    \label{fig:success_plot}
\end{figure}
```

## 🔧 更多自訂選項

如果您想進一步調整，可以修改 `lib/test/analysis/plot_results.py`:

### 1. 字體大小
```python
# 第 107-110 行
font_size = plot_opts.get('font_size', 20)          # 一般文字
font_size_axis = plot_opts.get('font_size_axis', 20)  # 軸標籤
font_size_legend = plot_opts.get('font_size_legend', 14)  # 圖例（已調整）
```

**建議值**:
- `font_size`: 16-20 (一般文字)
- `font_size_axis`: 18-22 (軸標籤)
- `font_size_legend`: 12-14 (圖例)

### 2. 線條寬度
```python
# 第 109 行
line_width = plot_opts.get('line_width', 2)
```

**建議值**: 1.5-3

### 3. 圖例位置
```python
# 第 155 行
ax.legend(..., loc=legend_loc, ...)
```

**可選位置**:
- `'lower left'` - 左下（Success Plot 預設）
- `'lower right'` - 右下（Precision Plot 預設）
- `'upper left'` - 左上
- `'upper right'` - 右上
- `'best'` - 自動選擇最佳位置

### 4. 圖表解析度
```python
# 第 169 行
fig.savefig(..., dpi=300, ...)
```

**建議值**:
- 螢幕查看: 150-200 dpi
- 論文發表: 300-600 dpi
- 預設: 300 dpi ✓
