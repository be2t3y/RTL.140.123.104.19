## 📋 下一步操作（兩種選擇）

### 🎯 選項 A: 測試現有模型（推薦）

如果您想快速測試模型效果，請按照以下步驟：

#### 步驟 1: 下載訓練好的模型權重

從 Google Drive 下載（三個版本任選一個）：
- 🔗 [SGLATrack-DeiT*](https://drive.google.com/drive/folders/1sHL7aFVZFwkPy6js48x-EKfoZC7oJc9X) - 推薦，性能最佳

下載後放置到：
```bash
output/checkpoints/train/sglatrack/deit_distilled/sglatrack_ep0297.pth.tar
```

建立目錄：
```bash
mkdir -p output/checkpoints/train/sglatrack/deit_distilled/
```

#### 步驟 2: 下載測試資料集

從百度網盤下載測試資料：
- 🔗 [測試資料集](https://pan.baidu.com/s/1MaeGLRcAUbJxksbF_CrOeQ?pwd=5vbv)
- 📝 提取碼: `5vbv`

解壓到 `data/` 目錄下（至少需要一個測試集）：
- `UAV123/` - 推薦，最常用的無人機追蹤資料集
- `UAV123_10fps/`
- `uavdt/`
- `V4RFlight112/` (UAVTrack112)
- `DTB70/`

#### 步驟 3: 執行測試

```bash
# 啟用環境
conda activate sgla

# 測試 UAV123
python tracking/test.py \
  --tracker_param sglatrack \
  --dataset uav123 \
  --threads 8 \
  --num_gpus 4

# 分析結果
python tracking/analysis_results.py
```

**預期結果** (SGLATrack-DeiT*):
- UAV123 AUC: 66.9%
- UAV123_10fps AUC: 65.5%

### 🔧 選項 B: 從頭訓練模型

如果您想要重新訓練或微調模型：

#### 步驟 1: 準備訓練資料集

下載並整理以下資料集到 `data/` 目錄：

**必需的訓練資料集**:
1. **LaSOT** - Large-scale Single Object Tracking
2. **GOT-10k** - Generic Object Tracking 10k
3. **COCO** - Common Objects in Context
4. **TrackingNet** - Large-scale Tracking Dataset

資料集結構：
```
data/
├── lasot/
│   ├── airplane/
│   ├── basketball/
│   └── ...
├── got10k/
│   ├── train/
│   ├── val/
│   └── test/
├── coco/
│   ├── annotations/
│   └── images/
└── trackingnet/
    ├── TRAIN_0/
    ├── TRAIN_1/
    └── ...
```

#### 步驟 2: 執行訓練

```bash
# 啟用環境
conda activate sgla

# 單 GPU 訓練
python tracking/train.py \
  --script sglatrack \
  --config deit_distilled \
  --save_dir ./output \
  --mode single \
  --use_wandb 0

# 或 4 GPU 訓練（更快）
python tracking/train.py \
  --script sglatrack \
  --config deit_distilled \
  --save_dir ./output \
  --mode multiple \
  --nproc_per_node 4 \
  --use_wandb 0
```

**訓練參數**:
- Epochs: 300
- Batch Size: 32
- Learning Rate: 0.0004
- 每輪樣本數: 60,000

訓練完成後，模型會保存在：
```
output/checkpoints/train/sglatrack/deit_distilled/sglatrack_ep{XXXX}.pth.tar
```
