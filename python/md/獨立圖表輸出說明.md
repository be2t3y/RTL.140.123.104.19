## 🔧 技術細節

### 修改位置
**檔案**: `tracking/calculate_metrics.py` (第 31-98 行)

### 主要改動

#### 1. 分別建立圖表
```python
# 原本：一張圖包含兩個子圖
fig, axes = plt.subplots(1, 2, figsize=(16, 6))

# 修改後：兩張獨立的圖
fig1, ax1 = plt.subplots(figsize=(10, 8))  # Success Plot
fig2, ax2 = plt.subplots(figsize=(10, 8))  # Precision Plot
```

#### 2. 分別保存
```python
# Success Plot
success_file = os.path.join(result_path, f'{dataset_name}_success_plot.pdf')
plt.savefig(success_file, dpi=300, bbox_inches='tight', format='pdf')
plt.close(fig1)

# Precision Plot
precision_file = os.path.join(result_path, f'{dataset_name}_precision_plot.pdf')
plt.savefig(precision_file, dpi=300, bbox_inches='tight', format='pdf')
plt.close(fig2)
```

#### 3. 更大的圖表
- **原本**: 16x6 (寬橫向，兩個圖並排)
- **修改後**: 10x8 (每張圖都是標準比例)

#### 4. 更高解析度
- **原本**: 150 dpi (螢幕品質)
- **修改後**: 300 dpi (論文品質)

#### 5. PDF 格式
- **原本**: PNG (點陣圖)
- **修改後**: PDF (向量圖，無限縮放不失真)
