## 🔍 兩種 FPS 的對比

| 指標 | 理論速度 | 實際速度 | 差異 |
|------|---------|---------|------|
| **FPS** | **416.99** | **187.38** | 2.22x |
| 延遲 | 2.40 ms | 5.34 ms | 2.22x |
| 測試數據 | 合成數據 | 真實圖像 | - |
| 包含 I/O | ❌ | ✅ | - |

### 📉 速度差異原因

**理論 FPS (416.99) > 實際 FPS (187.38)** 是正常的！

差異主要來自：
1. **圖像載入** (~30-40% 時間)
2. **預處理** (resize, normalize)
3. **後處理** (bbox 解碼、NMS)
4. **結果保存**
5. **第一幀初始化**

**效率比**: 187.38 / 416.99 = **44.9%**

這個效率比是合理的，表示約 55% 的時間用於 I/O 和後處理。
